# hyperterm-base16-tomorrow-dark
Hyperterm Theme based on Atom's `Base16 Tomorrow Dark` theme

How to use
----------
Add `hyperterm-base16-tomorrow-dark` to `plugins` in `~/.hyperterm.js`.
